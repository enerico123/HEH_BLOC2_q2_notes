## Rack 

Rack d'un switch 
et d'un routeur (Routeur Cisco C1111-4P)

## Config de Base du Routeur

Console (câble RJ45/DB9 – PuTTY Serial, 9600 baud)

1. Connexion initiale
Au démarrage, le routeur était vierge et a posé deux questions :

```cisco
Would you like to enter the initial configuration dialog? [yes/no]: no
Would you like to terminate autoinstall? [yes]: yes
```

2. Passage en mode privilégié

```cisco
Router> enable
Router#
```

3. Configuration de base

```cisco
Router# configure terminal

! Nom de l'équipement
Router(config)# hostname AGENCE-1

! Mot de passe enable (mode privilégié)
AGENCE-1(config)# enable secret <mot_de_passe>

! Mot de passe console
AGENCE-1(config)# line console 0
AGENCE-1(config-line)# password <mot_de_passe>
AGENCE-1(config-line)# login
AGENCE-1(config-line)# exit

! Retour en mode privilégié
AGENCE-1(config)# end

4. Sauvegarde de la configuration
AGENCE-1# write memory

Résultat : [OK] — configuration sauvegardée en NVRAM (survivra à un reboot).
```

5. Vérification

```cisco
! Vérifier la config sauvegardée
AGENCE-1# show startup-config

! Vérifier la config active
AGENCE-1# show running-config
```

## SSH 

**Équipement :** Routeur Cisco C1111-4P – RT-AGENCE1-G1
**Objectif :** Configurer le VLAN 70 (Gestion) et activer l'accès SSH
 
---
 
## 1. Création du VLAN 70
 
```
RT-AGENCE1-G1(config)# vlan 70
RT-AGENCE1-G1(config-vlan)# name Gestion
RT-AGENCE1-G1(config-vlan)# exit
```
 
---
 
## 2. Configuration de l'interface VLAN 70
 
```
RT-AGENCE1-G1(config)# interface vlan 70
RT-AGENCE1-G1(config-if)# ip address 172.20.70.1 255.255.255.0
RT-AGENCE1-G1(config-if)# no shutdown
RT-AGENCE1-G1(config-if)# exit
```
 
---
 
## 3. Assignation du port physique au VLAN 70
 
Le port **Gi0/1/0** (port 0 de la section 0/1/X) est dédié à la gestion.
 
```
RT-AGENCE1-G1(config)# interface GigabitEthernet0/1/0
RT-AGENCE1-G1(config-if)# switchport mode access
RT-AGENCE1-G1(config-if)# switchport access vlan 70
RT-AGENCE1-G1(config-if)# no shutdown
RT-AGENCE1-G1(config-if)# exit
```
 
---
 
## 4. Configuration SSH
 
```
! Domain name
RT-AGENCE1-G1(config)# ip domain name maroc.lan
 
! Génération des clés RSA 2048 bits
RT-AGENCE1-G1(config)# crypto key generate rsa modulus 2048
 
! SSH version 2 uniquement
RT-AGENCE1-G1(config)# ip ssh version 2
 
! Utilisateur local
RT-AGENCE1-G1(config)# username admin privilege 15 secret <mot_de_passe>
 
! Lignes VTY – SSH uniquement, pas Telnet
RT-AGENCE1-G1(config)# line vty 0 4
RT-AGENCE1-G1(config-line)# transport input ssh
RT-AGENCE1-G1(config-line)# login local
RT-AGENCE1-G1(config-line)# exit
```
 
---
 
## 5. Sauvegarde
 
```
RT-AGENCE1-G1# write memory
```
 
---
 
## 6. Vérifications
 
```
! Vérifier les VLANs
RT-AGENCE1-G1# show vlan brief
 
! Vérifier les interfaces et leurs IPs
RT-AGENCE1-G1# show ip interface brief
 
! Vérifier l'état d'un port
RT-AGENCE1-G1# show interfaces GigabitEthernet0/1/0
```
