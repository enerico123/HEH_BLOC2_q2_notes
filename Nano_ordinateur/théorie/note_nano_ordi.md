# Nano-ordinateur

## Système embarqué



>Définition :
Ordinateur dédié, intégré dans un système non informatique pour une tâche spécifique

exemples :

\- feu rouges

\- machine à laver

\- voiture

## Micro-controlleur \& nano-ordinateur

|Micro-controlleur|nano-ordinateur<br />|
|-|-|
|Arduino Uno|Rasberry pi|
|injecter code puis il va s'executer jusqu'à la mise hors tension |mini ordinateur complet <br />(carte mere, ram, processeur) |
|gère un seul module| peut gérer plusieurs module|



## Démonstration Rasberry pi

### 1. Se connecter au RPi
### 2. Écrire un programme Python pour allumer et éteindre la LED
### 3. Connecter une LED au RPi
### 4. Exécuter le programme (python ou autre)
