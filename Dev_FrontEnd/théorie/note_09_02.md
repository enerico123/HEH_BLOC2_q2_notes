# jQuery

## Intro

* jQuery : permet beaucoup d'animations
* exemples : carrousel, animations, load images...
* jQuery pas un language mais une bibliothèque/framework
* dans 80% des sites

## Mise en place

* télécharger biblio (jquery.com)
* version lisible : jquery.js
* CDN : réseau de diffusion de contenu
* version au dessus de 3.6
* console montre les erreurs
* $ is not defind erreur courante -> mauvais lien fichier
* $(function(){}) -> lancé au chargement du DOM
* $(document) -> passer un élélement HTML

## Sélections

* $() peut prendre 2 parametres -> cherche le premier élément dans le deuxième
* 6 Méthodes pour jouer sur les éléments

  * $('.bibou').parent()
  * ...

* Tableaux :

  * $('header')\[0] -> premier header



## Manipulation

Accéder/Modifier Attribus

* Lire contenu d'un élément -> .html()
* Modifier contenu d'un élément -> .html('Modif\_ici')
* Ajouter du contenu -> .append et autres
* Vider élément -> .empty() , .remove()
* Lire des attribus -> .attr('width')
* Modifier des attribus -> .attr('width','100px')
* Modifier plusieurs -> .attr({src='tata',width=100px})
* .prop -> valeur courante de l'attribu (attr est la valeur initiale)

Accéder aux classes

* .addClass()
* ...

Accéder aux positions

* distance entre -> .offset().left -> page
* distance entre -> .position().top -> parent

Accéder aux dimensions

* height()
* innerHeight()
* ...

Accéder aux valeur de formulaires

* .val() -> lire valeur
* .val('valeur') -> modifier valeur



## Écoute d'évènements

* .on('EVENT',FONCTION()) -> ecoute évenement
* .off('EVENT') -> N'ecoute plus l'évenement
* .trigger('EVENT') -> force le déclenchement de l'évenement
* $(document).ready(function{}) -> attends le chargement de la page
* $(this) -> fais référence à l'élément qui a déclenché la fonction

## Animation

* .fadeIn()
* .fadeOut()
* .animate({valeur css numérique}, 'fast', 'linear')
* .stop() -> stop la FILE d'animation de l'élement
* ...

ordre d'animation

* chaque élément a sa propre file d'attente
* possibilité d'attendre la fin d'une animation pour commencer une autre
